# Power BI Stuttering Treatment Dashboard

## Project Overview
This project analyzes patient stuttering treatment outcomes across multiple clinics and clinicians using interactive Power BI dashboards.

---

## Dashboard Overview

![Dashboard Overview](screenshots/dashboard-overview.png)

---

## Key Insights

- Medication B showed the highest average patient improvement.
- Site B demonstrated the strongest treatment outcomes across clinics.
- Longer treatment durations correlated with improved patient outcomes.
- Cognitive Therapy produced moderate improvement compared to medication-based treatments.
- Interactive filtering allows analysis by clinician, treatment type, site, and age group.
